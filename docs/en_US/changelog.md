# Changelog Speedtest by Ookla

- Translation of the plugin in English

## 04/12/2022

- Addition of a first template dedicated to the plugin mobile version (parameters in advanced equipment configuration => View tab => Widget section)

## 29/11/2022

- Addition of a first template dedicated to the Dashboard version plugin (parameters in advanced equipment configuration => View tab => Widget section)

## 09/11/2022

- First stable version

## 07/11/2022

- Addition of a cron dedicated to the plugin in the task engine
- Deletion of the Cron in the configuration of the plugin

- Correction of maximum time installation time
- Correction of the deletion of the closest server list to save equipment
- Correction of the server ID field type
- Deletion of the Add a command button in the command list

## 10/28/2022

- Addition of the closest server list in the equipment configuration
- Addition of the possibility of using a specific test server, to be used only if the server chosen automatically returns erroneous values

## 10/26/2022

- Deletion of the type column in the command list
- Deletion of the drop -down menu for binding action commands at info controls in the list of commands
- Historization of Download, Upload and Ping commands by default
- Addition of an option to deactivate errors when your Internet connection is out of service

## 25/10/2022

- Correction of the installation of outbuildings on Raspbian/Buster
